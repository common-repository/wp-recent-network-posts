== Description ==

WP Recent Network Posts adds a widget to Displays latest posts + thumbnails from selected Wordpress Network Blogs .You can add multiple blogs, choose how many posts you want to display and change style easily.

The main features are:

*	Widget Title
*   Number of posts of each blog to display
*	Thumbnails size
* 	List of blogs from the network to display
*	Show titles of blogs?
*	Give credit to author
*	Change Style Link

This plugin its very basic as it was built for a client. If you like it and you want to see more features like the ability to sort post by date, more commented ones, etc just let me know and i will add some features.

== Installation ==

1. Upload the entire `wp-recent-network-post` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find the new widget in your WordPress widgets section.
